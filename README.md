# 🎮 Pokemon Info App

Bem-vindo ao meu projeto **Pokemon Info**, criado com React!  
Este site sorteia um Pokémon aleatório ao carregar ou atualizar a página e exibe suas principais informações, como:

- ✅ Nome
- ✅ Tipos
- ✅ Altura e peso
- ✅ Habilidades
- ✅ Estatísticas base
- ✅ Imagem oficial
